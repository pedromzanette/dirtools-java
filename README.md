# DirTools - Java

## 📌 Sobre o projeto
Projeto desenvolvido em Java para manipulação de diretórios, utilizando recursão e busca em profundidade (DFS).

## 🚀 Funcionalidades
- Buscar arquivos por nome
- Listar diretórios
- Calcular tamanho de pastas
- Navegação em árvore de diretórios

## 🧠 Conceitos utilizados
- Recursão
- DFS (Depth First Search)
- Manipulação de arquivos em Java (File API)

## ▶️ Como executar

### Compilar:
```bash
javac src/DirTools.java
